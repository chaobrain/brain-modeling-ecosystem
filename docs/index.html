Why the BrainX ecosystem?

BrainX is a comprehensive software ecosystem that deeply integrates brain simulation with artificial intelligence technologies.

Compared to traditional brain simulators, it offers several key advantages:

  • Full-scale modeling: Dendritic computation → Point neuronal dynamics → Population models → Whole-brain models.
  • Differentiable optimization: Enables online/offline gradient-based optimization for large-scale model parameters.
  • High scalability: Code once, run on CPUs, GPUs, and TPUs, with ongoing expansion to neuromorphic hardware.
  • Powered by JAX: Deeply integrated with JAX and compatible with its extensive AI ecosystem.

Ecosystem components

Full-Scale Simulation

This ecosystem enables comprehensive brain simulation across multiple scales and modeling approaches. Our platform supports a diverse range of computational models, from detailed single-neuron representations to large-scale network architectures:

  • Morphology-based biophysically detailed neuron models
  • Hodgkin-Huxley point neuron models
  • Spiking network models
  • Population-level neural mass models
  • Whole-brain-scale models

This multi-scale approach allows researchers to investigate neural phenomena from molecular mechanisms to systems-level behavior, providing unprecedented flexibility for computational neuroscience research.

AI Compatibility

This ecosystem seamlessly integrates with modern artificial intelligence frameworks, bridging biological brain simulation and machine learning applications.

Users can directly incorporate machine learning libraries—such as Flax, Optax, Orbax, and many others—into their brain simulation workflows, enabling hybrid models, advanced optimization, and scalable experimentation.

Differentiable Optimization

This ecosystem supports advanced differentiable optimization techniques, enabling gradient-based learning and parameter tuning directly within biologically realistic neural simulations. Our platform seamlessly combines classical optimization methods with cutting-edge differentiable programming approaches:

  • Backpropagation through time (BPTT)
  • Surrogate gradient methods
  • Online gradient algorithms implemented in BrainTrace

Powered by JAX

Built on JAX, this ecosystem leverages high-performance numerical computing and automatic differentiation to enable efficient brain modeling at scale. JAX's GPU/TPU acceleration and just-in-time compilation make it possible to simulate large-scale neural networks with millions of neurons, while its composable function transformations allow seamless integration of brain models with modern AI frameworks.
